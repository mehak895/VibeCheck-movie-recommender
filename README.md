# CineSense – Movie Recommendation System

CineSense is a content-based movie recommendation system built to demonstrate **Data Structures & Algorithms**, **clean logic**, and **modular backend design**.  
The project focuses on explainable recommendation logic rather than black-box models.

---

## ✨ Features
- Content-based movie recommendations using similarity scoring  
- Fast lookup and filtering using efficient data structures  
- Modular code structure for easy extension and maintenance  
- Simple interactive UI for demonstration  

---

## 🧠 Core Concepts
- Data Structures: Arrays, Hash Maps  
- Algorithms: Similarity scoring, sorting, ranking  
- Design: Separation of logic, reusable components  
- OOP: Encapsulation and clean abstractions  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** pandas, numpy, scikit-learn  
- **UI (Demo):** Streamlit  
- **Version Control:** Git, GitHub
